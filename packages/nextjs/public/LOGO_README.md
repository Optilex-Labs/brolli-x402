# Optilex Logo Files for Twitter

## Profile Picture Options (400x400px)

### 1. `optilex-logo-twitter.svg` (Dark Background)
- Black background (#0a0a0a)
- Indigo logo (#4f46e5)
- Best for: Profile picture with automatic contrast

### 2. `optilex-logo-twitter-light.svg` (Light Background)  
- White background (#ffffff)
- Indigo logo (#4f46e5)
- Best for: Light mode displays

### 3. `optilex-logo-twitter-gradient.svg` (Brand Gradient)
- Navy to indigo gradient background
- White logo elements
- Best for: Premium brand look, matches website aesthetic

## Usage Instructions

### For Twitter Profile Picture:
1. Download your preferred SVG file
2. Upload directly to Twitter (Twitter supports SVG)
3. OR convert to PNG using:
   - **Online:** https://cloudconvert.com/svg-to-png (set to 400x400px)
   - **macOS Preview:** Open SVG → Export as PNG → 400x400px
   - **Command line:** `rsvg-convert -w 400 -h 400 optilex-logo-twitter.svg -o logo.png`

### Recommended:
Use **optilex-logo-twitter-gradient.svg** for your profile picture - it matches your site's brand aesthetic and looks professional at all sizes.
